1️⃣ What is the difference between var, let, and const?

In JavaScript, var, let, and const are used to declare variables. The main difference is how they behave in scope and reassignment. Var is function scoped and it can be redeclared and updated. Let is block scoped and it can be updated but cannot be redeclared in the same scope. Const is also block scoped but its value cannot be reassigned after it is declared. Because of these reasons, modern JavaScript usually prefers using let and const instead of var.


2️⃣ What is the spread operator (...)?

The spread operator (...) is used to expand elements of an array or object. It helps to copy or combine data easily. For example, we can use it to merge two arrays together or to create a copy of an existing array. It makes the code shorter and easier to understand.


3️⃣ What is the difference between map(), filter(), and forEach()?

Map, filter, and forEach are array methods used to work with array data. The map method creates a new array by transforming each element of the original array. The filter method also creates a new array but it only includes elements that match a certain condition. On the other hand, forEach simply loops through the array and performs an action for each element, but it does not return a new array.

4️⃣ What is an arrow function?

An arrow function is a shorter way to write functions in JavaScript. It uses the arrow symbol (=>) to define a function. Arrow functions are often used to make the code more concise and readable, especially in modern JavaScript.

5️⃣ What are template literals?

Template literals are a way to write strings in JavaScript using backticks instead of single or double quotes. They allow us to easily include variables or expressions inside a string. This makes it easier to create dynamic text and improves readability when working with strings.
